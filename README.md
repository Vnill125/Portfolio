## 🛠️ Getting Started

If you want to run the project locally:


# 1. Clone the repo
```bash
git clone https://github.com/Vnill125/Portfolio.git
```
# 2. Navigate into the project folder
```bash
cd Portfolio
```

# 3. Install dependencies
```bash
npm install
```

# 4. Start the development server
```bash
npm run dev
```
